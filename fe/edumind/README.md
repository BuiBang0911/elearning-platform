# Elearning Platform – Frontend

Frontend cho hệ thống Elearning, xây dựng bằng **React + Vite + Tailwind CSS**.

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Router

## Cài đặt

```bash
npm install
```

## Chạy dự án

```bash
npm run dev
```

Truy cập: http://localhost:5173

## Build production

```bash
npm run build
```
